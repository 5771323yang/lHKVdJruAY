# 项目简介

## 核心功能
- 基于YOLOv8的交通标志与行人车辆检测系统
- 支持1500张标注数据集，包含交通信号灯、停止信号、限速信号、人行横道信号、人行横道、行人、公交车、汽车、卡车等8类
- 提供完整源码、训练好的模型及训练结果、完整标注数据集和项目运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件和本地摄像头
- 界面功能：实时显示目标位置、目标总数和置信度信息
- 支持自定义检测参数：置信度阈值和IOU阈值
- 提供批量检测功能，可查看所有检测结果
- 支持检测结果保存功能
- 可选择显示检测标签和原图
- 提供多种模型选择选项

## 适用场景
- 交通监控系统
- 智能驾驶辅助系统
- 行人安全监测
- 交通流量分析
- 车辆识别与统计

## 技术特点
- 基于YOLOv8深度学习模型
- 使用Python3.9开发环境
- 依赖库：opencv-python、PyQt5、ultralytics、torch1.9
- 提供图形用户界面，操作便捷
- 支持多种检测模式和参数调整
- 兼容Pycharm+Anaconda或VSCode+Anaconda开发环境
